# 🧩 Игра Лабиринт

Классическая игра-лабиринт с несколькими уровнями сложности и современным дизайном.

## 🎯 Особенности

- **4 уровня сложности**: Легко, Средне, Сложно, Экстремально
- **Генерация лабиринтов**: Каждый лабиринт генерируется случайным образом
- **Система уровней**: Прогрессия сложности с каждым уровнем
- **Статистика**: Отслеживание времени и количества ходов
- **Современный дизайн**: Красивый интерфейс с градиентами и анимациями
- **Адаптивность**: Работает на разных размерах экрана

## 🚀 Как запустить

1. Откройте файл `index.html` в любом современном браузере
2. Выберите уровень сложности
3. Используйте стрелки или WASD для перемещения
4. Дойдите до зеленого выхода, чтобы пройти уровень!

## 🎮 Управление

- **Стрелки** или **WASD** - перемещение игрока
- **🔄 Начать заново** - перезапустить текущий уровень
- **📋 Меню** - вернуться в главное меню

## 📊 Уровни сложности

- **🌱 Легко**: 15x15 клеток - для начинающих
- **🌿 Средне**: 25x25 клеток - средняя сложность
- **🌳 Сложно**: 35x35 клеток - для опытных игроков
- **🔥 Экстремально**: 50x50 клеток - настоящий вызов!

## 🏆 Цель игры

Проведите красный шарик от начала лабиринта до зеленого выхода, используя как можно меньше ходов и времени!

Приятной игры! 🎮
